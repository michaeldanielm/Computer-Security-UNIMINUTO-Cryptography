Cifrado de sustitución - cifrados classicos

Un cifrado de sustitución es un método para codificar [sustituir] cada carácter de texto sin formato en un carácter de texto cifrado

¡Aviso! No utilice esto como medio para cifrar datos personales o importantes.

Sobre este sitio

Este sitio fue creado para aprender y compartir algunos métodos diferentes de codificación de texto sin formato en texto cifrado ilegible. En el transcurso de miles de años, la gente ha tenido un deseo de privacidad y / o secreto. Este proyecto fue asumido para aprender métodos de cifrado simples que se han utilizado a lo largo de la historia. Si bien también permite para que cualquiera pueda escribir un mensaje y poder compartirlo con amigos o twittearlo al mundo.
